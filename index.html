Lachlan's GitHub Cheat Sheet

Want to host your website through GitHub but can't figure out basic computer skills? Lachlan has you covered!

Step 1: Install Git and initialize a repo

  1. Open the Terminal and type this in:
      2. git --version
  2. Great, now Git is installed! Let's initialize a repo.
  3. Create a folder on your Desktop and name it whatever you want.
  4. Open Terminal and type cd then drag your repo folder into the Terminal.
  5. Now, run this command in the terminal:
      6. git init

Step 2: Commit with Git

  1. Let Git know bout the files you've created by entering into the terminal:
      2. git add .
  2. If no error message occurs, you're in the clear. Now, commit by typing:
      3. git commit -am "[enter custom message]"
  3. Now change to the main branch by entering:
      4. git branch -M main

Step 3: Connect your local folder to GitHub

  1. Go onto GitHub and copy the line below from the GitHub blank repo page, which is what you see after creating a new repo:
      2. git remote add origin [get link from GitHub]
  2. Now that your changes are committed, push your changes onto GitHub by saying:
      3. git push origin main
  3. Congratulations! Everything should be in GitHub now!

Now you're all done! Enjoy your website!
